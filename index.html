Создай аккаунт на JSONBin
Зайди на jsonbin.io → нажми Sign Up → зарегистрируйся через Google или email.
2
Создай хранилище (Bin)
После входа нажми NEW BIN, в поле введи {} и нажми Create. В адресной строке будет URL вида .../b/64abc123... — это и есть твой Bin ID.
3
Скопируй API Key
В верхнем меню нажми на своё имя → API Keys → скопируй Master Key.
Не удалось подключиться. Проверь Bin ID и API Key.
